App.yaml for static pages
^^^^^^^^^^^^^^^^^^^^^^^^^

.. code:: yaml

    runtime: php55
    api_version: 1
    threadsafe: true

    handlers:
     - url: /
       static_files: www/index.html
       upload: www/index.html

     - url: /(.*)
       static_files: www/\1
       upload: www/(.*)
